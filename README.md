# AI Services for London SMBs

Professional landing pages with AI chatbot integration for London small businesses.

## Overview

This repository contains AI-enhanced websites for various London small businesses. Each site features:

- **Business Information**: Name, address, phone number in UK format
- **Services Offered**: Detailed service listings with pricing in GBP (£)
- **AI Chatbot Integration**: Interactive chatbot for customer inquiries
- **Contact/Booking Form**: Functional forms for lead generation
- **Professional Design**: Clean, responsive UK-focused design

## Features

### AI Chatbot
Each website includes a custom AI chatbot that can answer common questions about:
- Services and pricing
- Opening hours
- Location and directions
- Emergency contacts
- Booking information

### Contact Forms
- Lead capture forms
- Service request forms
- Appointment booking forms

### UK-Specific Elements
- UK phone number formats (020 xxxx xxxx)
- GBP pricing (£)
- British English content
- Local London area references

## Usage

Simply open any HTML file in a web browser to view the landing page.

## Customisation

To modify a business page:
1. Edit the HTML file
2. Update business details (name, address, phone)
3. Modify services and pricing
4. Adjust chatbot responses in the JavaScript section
5. Update form endpoint if needed
